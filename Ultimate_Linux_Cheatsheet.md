## ‚ö° 7 Linux Productivity Tips

| Tip              | Command        | What It Does                          |
|------------------|----------------|---------------------------------------|
| Forgot sudo      | `sudo !!`      | Repeats last command with sudo        |
| Open last file   | `vim !$`       | Opens last argument (e.g., new file)  |
| Search history   | `Ctrl + R`     | Interactive history search            |
| Fix typo         | `fc`           | Opens last command in editor          |
| Folder jumping   | `pushd /path` / `popd` | Save and return to folders   |
| Tree view        | `tree -L 2`    | Shows folder structure cleanly        |
| Previous folder  | `cd -`         | Switches to last directory            |

**Bonus Tip:**  
Use `grep something file` instead of `cat file | grep something` ‚Äî cleaner and faster.
